# Vote-Mississippi
PHP script that runs users through a step-by-step questionnaire and automatically fills out a first time Mississippi voter registration form.
---

### Requirments:
+ [pdftk server]([https://sourceforge.net/projects/pdftk-builder-enhanced/](https://pdftk-server.en.lo4d.com/windows))
+ php
+ webserver
